## Step 1: 更新

``` bash
$ sudo apt update -y
```

## Step 2: 安裝

``` bash
$ sudo apt install postgresql postgresql-contrib
```

## Step 3: 確認

``` bash
$ sudo systemctl status postgresql
```

## Step 4: 登入

``` bash
$ sudo -i -u postgres
$ psql
# \q
```
